# Genesis Core - Public Pages

This directory contains public-facing web pages and assets for the Genesis Core project.

## Structure

- `index.html` - Main landing page
- `about.html` - About the project
- `docs.html` - Documentation portal
- `downloads.html` - Download releases
- `assets/` - CSS, JavaScript, images, and other static assets
- `css/` - Stylesheets
- `js/` - JavaScript files
- `images/` - Images and graphics

## Development

To serve the pages locally:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js
npx http-server
```

Then visit http://localhost:8000

## Deployment

These pages can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Contributing

When adding new pages, ensure they:
- Are mobile-responsive
- Follow accessibility guidelines
- Include proper meta tags for SEO
- Link to relevant documentation
