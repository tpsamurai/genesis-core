About Genesis Core

Genesis Core is a comprehensive ontology for identification and access management. It provides a well-defined schema and query language (GeneQL) for building secure, scalable identity and access management systems.

Built with modern best practices and designed to be database-agnostic, Genesis Core helps you implement fine-grained access control with ease.

Getting Started

Install GeneQL in your project:

JavaScript/TypeScript

npm install @genesis/geneql

Python

pip install genesis-geneql

Quick Example

// JavaScript
const { GeneQL } = require('@genesis/geneql');

const geneql = new GeneQL({
  host: 'localhost',
  database: 'genesis'
});

// Check if user has access
const canAccess = await geneql.checkAccess(userId, resourceId, 'read');

// Grant permission
await geneql.grantAccess(userId, resourceId, 'write');
